# League Auto
A program to automate league of legends client tasks such as auto accepting queue and calling role/locking in champ for blind pick.

To run the script, download the repo and run the .exe named `league-auto`. Keep everything in the folder as is.

Upon running the executable, a prompt like this should show up.
